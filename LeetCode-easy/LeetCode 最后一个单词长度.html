给定一个仅包含大小写字母和空格 ' ' 的字符串，返回其最后一个单词的长度。
如果不存在最后一个单词，请返回 0 。
说明：一个单词是指由字母组成，但不包含任何空格的字符串。
示例:
输入: "Hello World"
输出: 5




思路：切除前后空格防止干扰，然后切割，从右边获得最后一个字符串，返回字符串长度

代码：
def lengthOfLastWord(s):
   word = s.rstrip().split(' ')[-1]
   if s == ' ':
      return 0
   else:
      return len(word)
