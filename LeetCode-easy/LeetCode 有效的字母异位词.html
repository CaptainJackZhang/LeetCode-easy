给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的一个字母异位词。
示例 1:
输入: s = "anagram", t = "nagaram"
输出: true

示例 2:
输入: s = "rat", t = "car"
输出: false

说明:

你可以假设字符串只包含小写字母。

进阶:

如果输入字符串包含 unicode 字符怎么办？你能否调整你的解法来应对这种情况？

代码：
class Solution(object):
    def isAnagram(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """
        dict = {}
        for alpha in s:
            if dict.has_key(alpha):
                dict[alpha] += 1
            else:
                dict[alpha] = 1
        for alpha in t:
            if dict.has_key(alpha):
                dict[alpha] -= 1
            else:
                return False
        for i in dict.values():
            if i != 0:
                return False
        return True

优化：
直接使用集合，比较集合的字母，以及出现的次数
优化代码：
class Solution(object):
    def isAnagram(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """
        if len(s)!=len(t):
            return False
        st=set(s)
        for i in st:
            if s.count(i)!=t.count(i):
                return False
        return True
