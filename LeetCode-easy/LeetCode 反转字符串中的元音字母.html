编写一个函数，以字符串作为输入，反转该字符串中的元音字母。
示例 1:
输入: "hello"
输出: "holle"

示例 2:
输入: "leetcode"
输出: "leotcede"

说明:

元音字母不包含字母"y"。


思路：左右指针，往中间走。判断如果两个都是元音字符就交换，走到中间停止

代码：
class Solution(object):
    def reverseVowels(self, s):
        """
        :type s: str
        :rtype: str
        """
        yuanyin = ['a', 'e', 'i', 'o', 'u','A','E','I','O','U']
        start = 0
        lis_str=list(s)
        end = len(lis_str) - 1
        while (start <= end):
            if lis_str[start] not in yuanyin:
                start += 1
            elif lis_str[end] not in yuanyin:
                end -= 1
            else:
                tmp = lis_str[start]
                lis_str[start] = lis_str[end]
                lis_str[end] = tmp
                start+=1
                end-=1
        return ''.join(lis_str)
