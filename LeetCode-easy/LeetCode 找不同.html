给定两个字符串 s 和 t，它们只包含小写字母。字符串 t 由字符串 s 随机重排，然后在随机位置添加一个字母。请找出在 t 中被添加的字母。
示例:
输入：
s = "abcd"
t = "abcde"

输出：
e

解释：
'e' 是那个被添加的字母。


思路：
对两个字符串排序然后比较多出来的字符在哪里

代码：
class Solution(object):
    def findTheDifference(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: str
        """
        s1 = list(s)
        s1.sort()
        l1 = list(t)
        l1.sort()
        i = 0
        while (i < len(s1)):
            if s1[i] != l1[i]:
                return l1[i]
            i += 1
        return l1[-1]
