不转换为字符方法：
def isPalindrome(x):
   n = x
   newed = 0
   while (x > 0):
      newed = newed * 10 + x % 10;
      x /= 10;
   if (n == newed):
      return True
   else:
      return False
直接计算一个相反的数字出来判断是否相同
