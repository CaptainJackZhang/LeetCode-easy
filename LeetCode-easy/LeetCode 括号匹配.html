给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。
有效字符串需满足：
  1. 左括号必须用相同类型的右括号闭合。
  2. 左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。
示例 1:
输入: "()"
输出: true

示例 2:
输入: "()[]{}"
输出: true

示例 3:
输入: "(]"
输出: false

示例 4:
输入: "([)]"
输出: false

示例 5:
输入: "{[]}"
输出: true

思路：使用栈来存储括号，
如果是左括号，入栈，如果是右括号，判断栈如果为空，则为False,否则查看是否匹配，如果匹配，就从栈弹出该元素。
最后判断栈是否为空，如果为空，则为True
代码：
def isValid(s):
   stack=[]
   for ss in s:
      if ss in '([{':
         stack.append(ss)
      if ss in ')]}':
         if len(stack)<=0:
            return False
         else:
            compare=stack.pop()
            if (compare=='(' and ss!=')') or (compare=='[' and ss!=']') or(compare=='{' and ss!='}'):
               return False
   if len(stack)==0:
      return True
   else:
      return False
