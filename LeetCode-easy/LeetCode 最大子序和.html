给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。
示例:
输入: [-2,1,-3,4,-1,2,1,-5,4],
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。

进阶:
如果你已经实现复杂度为 O(n) 的解法，尝试使用更为精妙的分治法求解。

思路：
1.暴力破解，时间复杂度太高
2.从当前数字开始往后加，寻找最大值（超时）

def maxSubArray(nums):
   n = len(nums)
   ans = -999999999999
   for i in range(0, n):
      sum = 0
      for j in range(i, n):
         sum += nums[j]
         if sum > ans:
            ans = sum
   return ans
 

3.逐渐往后累加，当加和小于0时便抛弃
n = len(nums)
sum = 0
ans = -999999999999
if n <= 0: return 0
for i in range(0, n):
   if sum < 0:
      sum = nums[i]
   else:
      sum += nums[i]
   if sum > ans:
      ans = sum
return ans
4.大神优化版本
class Solution(object):
    def maxSubArray(self, nums):
        summ = 0
        max_sum = nums[0]
        for i in nums:
            summ+=i
            if max_sum < summ:
                max_sum = summ
            if summ < 0:
                summ = 0
        return max_sum
