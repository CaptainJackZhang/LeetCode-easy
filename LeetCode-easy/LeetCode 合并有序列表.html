将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 
示例：
输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4

思路：构造链表，同时递归比较，将值较小的放在前面

代码：
def mergeTwoLists(self, l1, l2):
   if not l1:
      return l2
   elif not l2:
      return l1
   else:
      if l1.val <= l2.val:
         l1.next = self.mergeTwoLists(l1.next, l2)
         return l1
      else:
         l2.next = self.mergeTwoLists(l1, l2.next)
         return l2

相当于两个链表互相插入重新改键
