给定一个非负整数组成的非空数组，在该数的基础上加一，返回一个新的数组。
最高位数字存放在数组的首位， 数组中每个元素只存储一个数字。
你可以假设除了整数 0 之外，这个整数不会以零开头。
示例 1:
输入: [1,2,3]
输出: [1,2,4]
解释: 输入数组表示数字 123。

示例 2:
输入: [4,3,2,1]
输出: [4,3,2,2]
解释: 输入数组表示数字 4321。

思路：先将列表转换为数字然后加一，最后在转换成列表
代码：
def plusOne(digits):
   num = digits[0]
   for i in range(0, len(digits) - 1):
      num = num * 10 + digits[i + 1]
   num += 1
   return map(int, str(num))
