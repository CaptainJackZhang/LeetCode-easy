给定一个非负索引 k，其中 k ≤ 33，返回杨辉三角的第 k 行。
在杨辉三角中，每个数是它左上方和右上方的数的和。
示例:
输入: 3
输出: [1,3,3,1]


思路：修改上一个程序，返回第n行：

代码：
def generate(numRows):
   numRows += 1
   if numRows == 1:
      return [1]
   tran = []
   i = 0
   while (i <= numRows):
      if i == 1:
         tran.append([1])
      if i == 2:
         tran.append([1, 1])
         lastlist = [1, 1]
      if i > 2:
         line = []
         line.append(1)
         j = 1
         while (j <= i - 2):
            line.append(lastlist[j - 1] + lastlist[j])
            j += 1
         line.append(1)
         lastlist = line
      i += 1
   return lastlist
