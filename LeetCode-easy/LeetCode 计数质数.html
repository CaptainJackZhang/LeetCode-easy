统计所有小于非负整数 n 的质数的数量。
示例:
输入: 10
输出: 4
解释: 小于 10 的质数一共有 4 个, 它们是 2, 3, 5, 7 。


思路:

我的代码:
class Solution(object):
    def countPrimes(self, n):
        """
        :type n: int
        :rtype: int
        """
        if n <= 2:
            return 0
        countnum = 0
        flag = [True] * (n - 1)
        flag[0] = False
        for cur in range(2, n):
            if flag[cur - 1] == False:
                continue
            i = cur * 2
            while (i < n):
                if flag[i - 1]:
                    flag[i - 1] = False
                i += cur
        for i in range(1, n):
            if flag[i - 1]:
                countnum += 1
        return countnum
