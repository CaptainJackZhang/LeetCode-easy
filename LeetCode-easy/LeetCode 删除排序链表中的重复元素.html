给定一个排序链表，删除所有重复的元素，使得每个元素只出现一次。
示例 1:
输入: 1->1->2
输出: 1->2

示例 2:
输入: 1->1->2->3->3
输出: 1->2->3

思路：按照链表的定义写程序就好，如果是相同元素就连接到他的下一个

代码：

def deleteDuplicates(head):
   if head is None:
      return head
   cur = head
   while cur.next:
      if cur.val == cur.next.val:
         cur.next = cur.next.next
      else:
         cur = cur.next
   return head
