给定一个非负整数 num，反复将各个位上的数字相加，直到结果为一位数。
示例:
输入: 38
输出: 2
解释: 各位相加的过程为：3 + 8 = 11, 1 + 1 = 2。 由于 2 是一位数，所以返回 2。

思路：
递归，取各位数字相加，结果小于10就返回


代码：

class Solution(object):
    def addDigits(self, num):
        """
        :type num: int
        :rtype: int
        """
        res = 0
        for i in str(num):
            res+=int(i)
        if res>=10:
            return self.addDigits(res)
        else:
            return res
