实现 int sqrt(int x) 函数。
计算并返回 x 的平方根，其中 x 是非负整数。
由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。
示例 1:
输入: 4
输出: 2

示例 2:
输入: 8
输出: 2
说明: 8 的平方根是 2.82842...,
     由于返回类型是整数，小数部分将被舍去。


思路：牛顿逼近法：
x1=x/2
X(n+1)=(Xn+a/Xn)/2
X(n+1)-Xn小于规定的精度时停止，返回整数结果

代码：
def mySqrt(x):
   if x <= 0:
      return 0
   wucha = 0.00001
   x1 = x / 2.0
   x2 = (x1 + x / x1) / 2.0
   while (abs(x1 - x2) > wucha):
      x1=x2
      x2 = (x / x1 + x1) / 2.0
   return int(x2)
