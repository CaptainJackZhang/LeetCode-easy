给定一个整数，编写一个函数来判断它是否是 2 的幂次方。
示例 1:
输入: 1
输出: true
解释: 20 = 1
示例 2:
输入: 16
输出: true
解释: 24 = 16
示例 3:
输入: 218
输出: false



思路:不断除2看是否有余数,有的话直接返回,最终判断商是不是1

代码:
class Solution(object):
    def isPowerOfTwo(self, n):
        """
        :type n: int
        :rtype: bool
        """
        if n == 0:
            return False
        while (n > 1) and (n % 2 == 0):
            n /= 2
        return (n == 1)
