给定一个正整数，返回它在 Excel 表中相对应的列名称。
例如，
    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB
    ...

示例 1:
输入: 1
输出: "A"

示例 2:
输入: 28
输出: "AB"

示例 3:
输入: 701
输出: "ZY"




思路：其实就是26进制的转换

代码：
def convertToTitle(n):
   result = ''
   if n < 26:
      return chr(ord('A') + n - 1)
   while (n > 26):
      yushu = n % 26
      if yushu == 0:
         result += 'Z'
         n-=26
      else:
         result += (chr(ord('A') + yushu - 1))
      n = n / 26
   result += (chr(ord('A') + n - 1))
   return result[::-1]

class Solution(object):
   def convertToTitle(self, n):
      """
      :type n: int
      :rtype: str
      """
      result, dvd = "", n

      while dvd:
         result += chr((dvd - 1) % 26 + ord('A'))
         dvd = (dvd - 1) / 26
      return result[::-1]
