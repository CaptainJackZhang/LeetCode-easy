给定一个非负整数 numRows，生成杨辉三角的前 numRows 行。
在杨辉三角中，每个数是它左上方和右上方的数的和。
示例:
输入: 5
输出:
[
     [1],
    [1,1],
   [1,2,1],
  [1,3,3,1],
[1,4,6,4,1]
]

思路：前两行都是1，从第三行开始，按照定义每行生成上一行的加和
代码：
def generate(numRows):
   tran = []
   i = 0
   while (i <= numRows):
      if i == 1:
         tran.append([1])
      if i == 2:
         tran.append([1, 1])
         lastlist = [1, 1]
      if i > 2:
         line = []
         line.append(1)
         j = 1
         while (j <= i - 2):
            line.append(lastlist[j - 1] + lastlist[j])
            j += 1
         line.append(1)
         lastlist=line
         tran.append(line)
      i += 1
   return tran
